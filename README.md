<h2 style=":center">--SETUP--</h2>
-First set up your database<br>
-Create .env file at server folder root and set it up as follow:<br>
<ul>
    <li>PORT=(server port)</li>
</ul>
    CLIENT_PORT
    DB_HOST
    DB_USER
    DB_PASS
    DB=(DB name)
    DB_DIALECT
    TKEY=(token key)
    URL_SERVER
    URL_CLIENT

-npm install in the server folder.
